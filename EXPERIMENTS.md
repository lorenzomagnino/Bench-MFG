# Experiments

For each environment: LasryLionsChain, NoInteractionChain, FourRoomsAversion2D, RockPaperScissors, SISEpidemic, KineticCongestion, MultipleEquilibriaGame, ContractionGame; change each variable ENVIRONMENT="LasryLionsChain" in each of the following script to run the environment selected.

```bash
#ATTENTION: before running it, select algorithm/damped_fixed_point in defaults.yaml
./scripts/run_dampedfp.sh

#ATTENTION: before running it, select algorithm/omd in defaults.yaml
./scripts/run_omd.sh

#ATTENTION: before running it, select algorithm/pi in defaults.yaml
./scritps/run_pi.sh

#ATTENTION: before running it, select algorithm/pso in defaults.yaml
./scripts/run_pso.sh
```
